## 关于我们

**板路声语实验室**（Banlu Voice Lab）是一个专注于桂林方言语音合成（TTS）研究的开源项目。

- "**板路**" 是桂林话中"讲故事"的意思
- 我们的目标是用现代 AI 技术，为桂林方言建立数字语音档案
- 技术优先：开源方案最低成本，优先文化传承而非商业变现

*板路，即桂林话"讲故事"。我们想把这些故事，用AI的声音讲下去。*
