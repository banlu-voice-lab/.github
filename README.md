## 关于我们

**板路声语实验室**（Banlu Voice Lab）是一个专注于桂林方言语音合成（TTS）研究的开源项目。

- "**板路**" 是桂林话中"讲故事"的意思
- 我们的目标是用现代 AI 技术，为桂林方言建立数字语音档案
- 技术优先：开源方案最低成本，优先文化传承而非商业变现

## 核心技术

| 技术栈 | 说明 |
|--------|------|
| **GPT-SoVITS** | 桂林话TTS微调框架 |
| **1-3小时录音** | 即可完成基础模型训练 |
| **开源优先** | 全流程可复现 |

## 当前项目

### [banlu-voice-lab](https://github.com/banlu-voice-lab/banlu-voice-lab)

桂林方言语音合成模型 — 基于 GPT-SoVITS 微调，支持桂林市区话（桂柳话）。

> ⚠️ 项目正在积极开发中，模型权重和文档陆续发布。

## 方言地图

桂林话属于**桂柳话**体系，主要分布于：

- 桂林市区
- 柳州
- 河池
- 贵州南部部分地区
